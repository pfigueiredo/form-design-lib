# Demo App

This is a demo application for testing the Form Design Library.

## Setup

1. First, build the library:
   ```bash
   cd ../form-design-lib
   npm install
   npm run build
   ```

2. Then, install dependencies for the demo app:
   ```bash
   cd ../demo-app
   npm install
   ```

3. Run the demo app:
   ```bash
   npm run dev
   ```

The app will be available at `http://localhost:5173`
